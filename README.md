# E-Commerce Sales Performance Dashboard

## Project Overview
This project features an interactive **Sales Performance Dashboard** designed to analyze transactional data and provide actionable business insights. The dashboard monitors key revenue drivers, customer engagement, and geographical sales distribution to support data-driven decision-making.

## Key Performance Indicators (KPIs)
The dashboard tracks four primary business metrics:
* **Total Revenue:** $9M - Overall gross sales generated.
* **Total Orders:** 22K - Total volume of successful transactions.
* **Active Customers:** 4K - Unique customer base engaged during the period.
* **Average Order Value (AOV):** 433.63 - The mean value of each transaction.

## Technical Features & Analysis

### 1. Revenue Trend Analysis
* **Time-Series Visualization:** A line chart displaying monthly revenue fluctuations.
* **Growth Insight:** Observed a significant upward trend from September 2011 ($0.7M) to a peak in November 2011 ($1.4M).

### 2. Product Performance
* **Top 5 Best-Sellers:** Ranked by total revenue contribution.
* **Leading Item:** "DOTCOM POSTAGE" identified as the top-performing product with 203K in revenue.

### 3. Geographical Distribution
* **Global Market Mapping:** A geospatial visualization showing sales concentration primarily in the European region.
* **Market Expansion:** Helps identify high-performing territories for targeted marketing.

### 4. Interactive Functionality
* **Dynamic Slicers:** Users can filter data by **Invoice Date** ranges and **Product** categories.
* **Drill-down Capabilities:** Interactive charts allow for granular analysis of specific data points.

## Tech Stack
* **Tool:** Power BI / Tableau.
* **Languages/Skills:** DAX (Data Analysis Expressions), Data Modeling, ETL, Geospatial Analysis.
* **Dataset:** E-commerce Transactional Data.
